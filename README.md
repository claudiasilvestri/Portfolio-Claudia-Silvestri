# 💻 Claudia Silvestri Portfolio

Personal portfolio website developed to showcase **Frontend** projects, technical skills, and professional experience.

The website is designed as a professional online presence for recruiters, clients, and networking opportunities, with a focus on accessibility, responsiveness, and clean user experience.

Built with React and Vite, it includes project highlights, services, downloadable CV, and contact information.

## ✨ Features

- Responsive design
- Professional portfolio presentation
- Project showcase
- Downloadable CV
- Contact section
- SEO optimization
- Custom domain integration

## 🛠️ Tech Stack

**Frontend**

- React
- Vite
- JavaScript
- HTML
- CSS

**Deployment**

- Vercel

## 🚀 Installation
npm install
npm run dev
